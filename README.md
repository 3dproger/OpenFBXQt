# OpenFBXQt
Lightweight engine for working with 3D graphics on Qt
- No Assimp or other additional dependencies
- Integrated with Qt 5 (can be adapted for Qt6)
- Skeletal animation
- FBX file support
- Works on Windows, Linux, macOS, Android, iOS and other
